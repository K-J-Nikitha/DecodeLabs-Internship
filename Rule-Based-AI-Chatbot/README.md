# Rule-Based AI Chatbot

## Introduction

This project is a simple chatbot developed using Python.

The chatbot responds to predefined user inputs using rule-based logic (if-elif-else).

## Objectives

- Learn AI basics
- Understand decision making
- Implement chatbot using Python

## Features

- Greeting messages
- AI related responses
- Python related responses
- Help menu
- Exit command
- Unknown message handling

## Technologies

- Python 3

## Run

python chatbot.py
