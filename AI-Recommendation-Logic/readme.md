# AI Recommendation Logic

This project was developed as **Week 3** of the DecodeLabs Backend Development Internship.

## Project Objective

Build a simple AI recommendation system that suggests items based on user preferences.

---

## Features

- User selects category
- User selects experience level
- Rule-Based AI Recommendation
- Responsive User Interface
- Flask Backend
- JavaScript Fetch API
- JSON Response

## Installation

Clone repository

```bash
git clone https://github.com/K-J-Nikitha/DecodeLabs-Internship.git
```

Go inside project

```bash
cd AI-Recommendation-Logic
```

Create virtual environment

```bash
python3 -m venv venv
```

Activate

Linux

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run application

```bash
python app.py
```

Open browser

```
http://127.0.0.1:5000
```
